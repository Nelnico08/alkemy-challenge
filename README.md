## Installation
***
A little intro about the installation. 
### Backend:

```
$ git clone https://github.com/Nelnico08/alkemy-challenge.git
$ cd ../path/to/the/file/client/api
$ npm install
$ npm start
```

### Frontend:
```
$ git clone https://github.com/Nelnico08/alkemy-challenge.git
$ cd ../path/to/the/file/client
$ npm install
$ npm start
```
